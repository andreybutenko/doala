About Doala

Doala is the to-do app for koala fans! The user experience is designed to speed up most workflows through keyboard shortcuts. Your lists are safely saved on your computer and are never shared.

Navigation

  • ArrowUp: move up to above todo item
  • ArrowDown: move down to below todo item
  • Ctrl + ArrowRight: move right to next todo list, creating a new list if necessary
  • Ctrl + ArrowLeft: move left to previous todo list

Operations

  • Enter: create new todo item
  • Ctrl + Enter: mark current todo item complete
  • Ctrl + Backspace: delete current todo item

Reorganizing

  • Ctrl + ArrowUp: move todo item up
  • Ctrl + ArrowDown: move todo item down
  • Ctrl + Shift + ArrowLeft: move todo list left
  • Ctrl + Shift + ArrowRight: move todo list right

Other

  • Rename a list by double-clicking its name
  • Ctrl + .: toggle dark mode
  • Ctrl + /: toggle this menu
